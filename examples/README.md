# Examples

This directory contains installation and integration examples for the supported agent runtimes.

- [claude-code.config.json](/C:/new/mcp-ToseaAI/examples/claude-code.config.json): local MCP registration for Claude Code style clients
- [cursor.mcp.json](/C:/new/mcp-ToseaAI/examples/cursor.mcp.json): Cursor MCP config
- [openai-agents-typescript.ts](/C:/new/mcp-ToseaAI/examples/openai-agents-typescript.ts): OpenAI Agents SDK example using `MCPServerStdio`

These are templates. Replace the absolute command path and environment variables for your environment before use.
